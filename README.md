# kb-color

Controls the backlight color and brightness of the Gigabyte Aorus 15P keyboard via HID on Linux.

## Build

```bash
make
```

## Install

```bash
make install
```

## Usage

```bash
kb-color --color red --brightness 100
kb-color -c blue -b 50
kb-color --brightness 30
kb-color --color white
kb-color --list
```
